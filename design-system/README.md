# Design System

VM Nexus uses one design language: calm enterprise surfaces, restrained accent colors, clear information density, and stable spacing.

Rules:

- Use one font family across the ecosystem.
- Use tokens from this folder before adding custom values.
- Use soft shadows, controlled blur, and subtle motion only where they improve focus.
- Keep all modules visually related across BusinessOS, Vidyaluma, VaanMeet, Pulse Forms, and VMetron.
