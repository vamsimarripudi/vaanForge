# Multi-Agent System

VaanForge coordinates specialized agents as a product team rather than relying on one generic assistant.

## Roles

- Product Manager Agent
- Architect Agent
- UI/UX Agent
- Backend Agent
- Frontend Agent
- QA Agent
- Security Agent
- DevOps Agent
- Documentation Agent

## Collaboration Records

- Role configuration
- Assignments
- Handoffs
- Agent comments
- Conflicts
- Decision logs
- Reviews
- Final reviews

## Final Review Policy

Product, Architecture, QA, Security, and DevOps reviews must pass before high-confidence completion. Human overrides are allowed only when audited.
