# Roles

Core roles:

Founder, Co-founder, CEO, CTO, COO, CFO, CMO, HR, Employee, Developer, Sales, Support, Legal, CA, Creator, Partner, Client, Customer, Viewer, Admin, Super Admin.

Founder and Super Admin control final permissions.

Every role must have:

- Permissions.
- Dashboard view.
- Allowed modules.
- Restricted actions.
- Audit logs.
