# Memory Engine

The memory engine stores learning signals only after review. It avoids automatic trust.

## Memory Types

- Project memory
- User preference memory
- Product-specific memory
- Error-fix memory
- Template usage memory
- Deployment memory
- Approval/rejection memory

## Safety

- Secret scanning before storage
- Prompt-injection scanning before storage
- Review-gated trust
- Archived/rejected memories excluded from retrieval
- Retrieval logs explain why suggestions were selected
