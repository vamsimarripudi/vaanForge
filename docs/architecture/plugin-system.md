# Plugin System

The plugin system is part of the KRAVIA Developer Platform and supports extension of agents, workflows, templates, validation, deployment, and event hooks.

## Plugin Requirements

- Registered developer account
- Manifest
- Permissions
- Review state
- Prompt-injection scan
- Audit trail

## Current Scope

The repository includes plugin registry surfaces and marketplace publishing foundations. External runtime sandboxing and plugin package execution should remain gated by security review.
