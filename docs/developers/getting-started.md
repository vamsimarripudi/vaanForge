# Developer Getting Started

The KRAVIA Developer Platform exposes controlled integration surfaces for VaanForge.

## Start

1. Sign in to the developer portal.
2. Create a developer app.
3. Create a scoped API key or OAuth-ready client.
4. Configure webhooks if you need event callbacks.
5. Register plugins or marketplace apps only after reviewing permissions and manifest content.

## Local Commands

```powershell
npm.cmd run kdp
npm.cmd run test:e2e
```
