# Testing

VaanForge uses service tests and repository-level QA contracts.

## Commands

```powershell
npm.cmd run lint
npm.cmd run typecheck
npm.cmd run test
npm.cmd run test:e2e
npm.cmd run build
```

## Contract Areas

- Route smoke
- UI interactions
- API security
- CSRF
- Database
- Environment
- Production readiness
- Infrastructure
- Documentation
