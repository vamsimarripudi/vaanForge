# Prediction Engine

Predictions are exposed through `/api/v1/admin/intelligence/predictions`.

## Predictions

- low credits
- storage exhaustion
- subscription renewal risk
- customer churn
- deployment risk
- infrastructure overload
- provider outage impact
- queue saturation
- database growth
- project completion delay

Each prediction includes risk level, confidence, horizon, reason, evidence, and recommended action.

The current engine is heuristic and transparent. It does not claim trained ML behavior.
