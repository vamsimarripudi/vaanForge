# Command Palette

The command palette provides keyboard-first access to common VaanForge actions.

Shortcut: `Ctrl/Cmd + K`

## Command Sources

- Projects
- Billing
- Factory
- Settings
- Documentation
- Support

## APIs

- `GET /api/v1/onboarding/command-palette`
- `POST /api/v1/onboarding/command-palette/run`

Command usage is stored in `commandUsage` for recent commands and adoption analytics.
