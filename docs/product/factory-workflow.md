# Factory Workflow

`factory.vaanforge.com` exposes the Autonomous Software Factory as a step-by-step workflow.

## Step Order

1. Chat about idea
2. Intake
3. Follow-up questions
4. Requirement quality score
5. Blueprint generation
6. Blueprint approval
7. Design direction
8. Task graph
9. Build start
10. File and diff review
11. QA and security
12. Deployment
13. Release and docs

Future steps remain locked by backend state. For example, blueprint generation is blocked until open follow-up questions are answered, and build start is blocked until design approval.
