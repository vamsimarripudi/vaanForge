# VaanForge Product Workflow

VaanForge is an AI software factory where a user moves from idea to validated release with conversational guidance and code execution.

## Core Flow

1. Create workspace and choose plan.
2. Create project.
3. Describe the idea in natural language.
4. Answer follow-up questions generated by the Requirement Agent.
5. Generate and approve blueprint.
6. Generate design direction and approve it.
7. Generate task graph.
8. Start agent execution.
9. Track files, diffs, QA, security, deployment, release, docs, and memory.
10. Deploy only after validation passes.

## UX Principles

- The app opens directly into the workspace.
- The sidebar contains history and profile controls.
- The main workspace focuses on the next action.
- No fake charts or static success states.
- Empty, loading, error, permission, plan-limit, and success states must be explicit.
