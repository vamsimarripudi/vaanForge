# Enterprise Trust Layer

The enterprise trust layer combines security telemetry, privacy controls, policy acceptance, API abuse controls, prompt-injection defense, secret scanning, and readiness reporting.

## Core Records

- `security_events`
- `security_risk_scores`
- `privacy_export_requests`
- `privacy_delete_requests`
- `policy_acceptance_logs`
- `audit_exports`
- `api_key_security_settings`
- `prompt_risk_events`
- `secret_scan_events`
- `security_reports`

## Operating Principle

Every sensitive action must be authenticated, authorized, tenant-scoped, validated, and audited.

## Compliance Language

VaanForge can produce readiness evidence. It must not claim external certifications until those certifications are legally obtained by KRAVIA PRIVATE LIMITED.
