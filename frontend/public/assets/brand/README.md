# VaanForge Brand Assets

This folder contains production-ready visual assets derived from the internal VaanForge brand design system.

Use these assets in product surfaces, PWA metadata, social cards, email templates, loading screens, and error states. The `/brand` route remains an internal design reference and is intentionally hidden from the product navigation.

## Structure

- `tokens.json` - canonical color, typography, radius, and usage metadata.
- `logos/` - mark and wordmark variants for dark, light, and primary surfaces.
- `icons/` - favicon, PWA icons, and touch icon assets.
- `splash/` - dark and light app launch screens.
- `social/` - Open Graph/social preview card.
- `states/` - loading, empty, success, error, and plan-limit state artwork.
- `collateral/` - email and document header assets.
